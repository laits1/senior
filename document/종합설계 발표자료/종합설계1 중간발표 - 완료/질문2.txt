1. 지오펜스를 어떻게 구현할 것인가??

유아 및 치매 환자가 착용하고있는 로라모듈과 연결되어있는 보호자의 애플리케이션 사이의 거리가 
보호자가 설정한 거리의 범위를 넘었을 경우 애플리케이션을 통해 알람을 오게 한다. 


2. 로라망을 어떻게 사용할 것인가?
현재 SKT에서 Lora 무료 모듈을 신청해 수령받았고 처음에는 이 모듈을 가지고 skt 분당 수내 사옥에
가서 '오픈 테스트 베드' 를 통해 테스트 망을 사용해야 한다. 
여기서 SKT의 40여가지 항목의 테스트 검증을 통과하면 상용망 시험허가를 받을 수 있다.
상용망 시험허가를 받으면 3개월간 무료로 전국 어디서든 무료로 사용을 할 수있다.
3개월이 지나면 SKT LoRa 상용망 서비스를 사용해야하며 이에 따른 이용 요금도 부과해야한다.
가장 비싼 요금제가 10MB에 1,100원 정도로 가격은 매우 저렴하다.


3. 기존에 있는 제품과 보여줄 수 있는 차이점?
기존에 있는 리니어블 팔찌 같은 제품은 블루투스로 통신을 하기 때문에 사용자와 매우 짧은 거리밖에
통신을 할 수 밖에없다. 
하지만 저희가 개발하는 제품은 LoRa 기반으로 저전력 광역통신을 사용하기 때문에 언제 어디서든 
거리가 아무리 멀더라도 실시간으로 위치를 확인할 수 있다는게 큰 차이이다.

4. ThingPlug가 구체적으로 무엇인가?
ThingPlug는 LoRa 네트워크 서버와 연결되어 LoRa 디바이스로부터 생성된 데이터를 관리하는 기능을 하며,
외부 엔티티의 LoRa 네트워크 서비스 접근을 위한 기능을 제공한다.
즉, LoRa 디바이스로부터 생성한 데이터는 우선 모두 ThingPlug로 송신이 되고, IoT 디바이스 개발자들이 개발을
하는데 기본적으로 필요한 기능들을 API 형태로 제공을 합니다.
(개발하는데 기본적으로 필요한 기능 : Device/APP Registration, Data Management, Data Monitoring, 
Connection Management, Device Management, Security, Location Management, Group Management)
